```shell
# 控制台输入 
chcp 65001

# 超级终端
$OutputEncoding = [Console]::OutputEncoding = [Text.Encoding]::UTF8
```

### vscode 快捷键
- 折叠当前代码块
  - Ctrl + Shift + [
- 展开当前代码块：
  - Ctrl + Shift + ]
- 折叠所有代码块
  - Ctrl + K, Ctrl + 0
- 展开所有代码块
  - Ctrl + K, Ctrl + J
